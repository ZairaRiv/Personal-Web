Projects

RNA-Sequencing

January 2019 - May 2019

Bioinformatics

This project started in January 2019 and recently finished in May 2019. It consisted of myself and a Professor to guide me along the process, where I took chromosome files from the UCSC human genome database and performed intron masking. This was a pipelined project that required the use of Bowtie (an analytics tool), as well as programs, are written in Python to help with the intron masking as well as the mapping of the genes that was done at the end. The amount of data sorted for the human genome was about 3 GB and was compared to patients files to be able to see how they varied and what kinds of things might be different between a sick and healthy human.

ESNS

August 2017 - May 2018

Front-end Developer

One of my longest running projects and the one that I am most proud of is this website https://esns.life/ which I designed and created along with the help of multiple other members. The purpose of the website is to be able to track down an on-campus school shooter. We worked using a combination of Scrum, XP and Agile method. This website was done in several languages and actually had several versions before this main one was finally established after a year-long working on it.

Nonogram

August 2018- December 2018

Front and backend developer

A semester-long project where users can interactively play nonogram. This was created using HTML, CSS with an original cowboy design as well as PHP and MySQL for the backend since users can create accounts and try to beat the best scores of other players.
